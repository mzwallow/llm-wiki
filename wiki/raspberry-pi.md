# Raspberry Pi

Raspberry Pi is a series of small single-board computers (SBCs) developed in the United Kingdom by the Raspberry Pi Foundation in association with Broadcom.

## Models
- **Raspberry Pi 1, 2, 3**: Legacy models with increasing performance.
- **Raspberry Pi 4**: Significant spec upgrade with USB 3.0 and high RAM options.
- **Raspberry Pi 5**: Current flagship with 2-3x performance boost over Pi 4 and new features like a power button and RTC.
- **Raspberry Pi Zero/Zero 2**: Minimalist, low-power boards for compact projects.

## Hardware Features
- **SoC**: Broadcom BCM series.
- **GPIO**: General-purpose input/output pins for hardware interfacing (I2C, SPI, UART).
- **Storage**: Typically boots from a micro SD card.
- **Connectivity**: Ethernet, WiFi, Bluetooth (model dependent).

## Use Cases
- Home Automation
- Networking (e.g., [[openwrt]])
- Learning Programming
- Media Centers
